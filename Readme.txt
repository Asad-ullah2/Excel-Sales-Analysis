# 📈 Excel-Based Sales Analytics Project

This repository presents a complete end-to-end sales analytics solution built in Microsoft Excel. It follows a structured approach to transforming raw transactional data into insightful visual dashboards using pivot tables and dynamic charting tools.

---

## 📦 Project Contents

- **`dataset.xlsx`**  
  Contains the original raw sales dataset, including fields such as region, product, date, and revenue.

- **`pivot-tables.xlsx`**  
  Features cleaned and structured data with Pivot Tables used for summarizing key performance indicators (KPIs) like total revenue, monthly trends, and regional comparisons.

- **`dashboard.xlsx`**  
  A fully interactive Excel dashboard integrating charts, slicers, and conditional formatting to enable quick insights and user-driven filtering.

---

## 🚀 Key Features

- ✔️ Clean and structured raw data for ease of analysis  
- ✔️ Multiple pivot tables for dynamic summaries  
- ✔️ Professionally designed dashboard layout  
- ✔️ Use of slicers for interactive filtering (e.g., by month or region)  
- ✔️ Charts for visual representation of sales trends and performance metrics  
- ✔️ Conditional formatting to highlight outliers and patterns

## 📊 Business Use Case

This project simulates a real-world scenario where a business needs to analyze its monthly sales performance across multiple regions and product lines. It is designed for quick decision-making by:

- Identifying best- and worst-performing regions
- Monitoring monthly revenue trends
- Visualizing performance through charts and KPI indicators
